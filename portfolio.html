Hand Signs Recognizer

This project uses MediaPipe Hands and OpenCV to recognize hand gestures in real-time. The system detects and tracks hand landmarks using MediaPipe, processes the hand position, and classifies gestures based on predefined signs. OpenCV handles image processing, while a machine learning model or rule-based logic interprets the signs. This can be used for applications like sign language recognition, touchless controls, and interactive interfaces.
